# friendly-octo-invention
A Data-Driven Approach to AI and Python

This project uses `uv` with Python 3.14 and includes Jupyter Notebook, pandas, and seaborn for data analysis.

## Getting started

```bash
uv sync
uv run jupyter notebook
```
